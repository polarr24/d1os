# Laptop — maximum D1 configuration

Everything is staged. Five moves, ~4 minutes, and the machine is transformed:

1. **Wallpaper**: right-click Desktop > Personalize > Browse >
   `ZAYDEN-D1/wallpaper-laptop-2560x1440.png`. Your name, the gate, your words.
2. **The app, one double-click**: `D1-OS.bat` on the Desktop opens ZAYDEN D1 OS
   in its own chromeless window — a real app, not a browser tab. Right-click it
   > Pin to taskbar (pin the window it opens, or drop the .bat into
   shell:startup to launch on boot).
3. **The widget**: `D1-WIDGET.bat` opens the glanceable face — gate countdown +
   beat-last-week — sized for a corner. Open it when you sit down; it stays up.
4. **Files have one home**: film, backups, coach emails, report cards —
   everything lands in `ZAYDEN-D1/` (README inside explains where). If it's
   career-related and it's loose on this laptop, it's in the wrong place.
5. **After you deploy** (you said you've got it): the phone + iPad + this laptop
   all run the same live app, and sync (SETUP-SYNC.md) makes them one brain.
   The laptop's copy in D1-OS.bat runs from the project file — it updates the
   moment you edit/rebuild, before the hosted URL even sees it.

Games stay. Nothing was deleted. The career layer just outranks them now.
